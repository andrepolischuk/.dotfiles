# .dotfiles

> Dotfiles for shell, zsh, etc

![](screenshot.png)

## Install

```sh
# dotfiles
git clone https://github.com/andrepolischuk/.dotfiles && ~/.dotfiles/init

# git config
git config --global user.name "Your Name"
git config --global user.email "email@yoursite.com"
```
