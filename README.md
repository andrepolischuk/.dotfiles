# .dotfiles

> Dotfiles for shell, zsh, etc

## Install

```sh
git clone https://github.com/andrepolischuk/.dotfiles && ~/.dotfiles/init
```

## Apps

* [zsh][zsh]
* [oh-my-zsh][oh-my-zsh]
* git
* [atom][atom]
* vim
